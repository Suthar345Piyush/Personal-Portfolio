## 🚀 Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/Suthar345Piyush/Piyush-sPortfolio.git

Navigate to the project directory
cd Piyush-sPortfolio

npm install
# or
yarn install

npm run dev
# or
yarn dev
